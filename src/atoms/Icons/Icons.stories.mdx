# Iconos

Los iconos ayudan a las personas a navegar de manera intuitiva y fácil

## Iconos principales

Los iconos principales son utilizados para identificar tipos de elementos y recursos relevantes para el alumno dentro de la plataforma.

**Tamaño:** 45 x 45 px

<ArrowRight />

## Iconos secundarios

Los iconos secundarios son los más utilizados en la plataforma, se utilizan para acompañar datos que entregan información relevante a los alumnos. En algunos casos también se utilizar para complementar la interacción.

**Tamaño:** 32 x 32 px

<ArrowRight />

## Iconos para hitos

Los hitos representan momentos destacables dentro del viaje de estudio de un alumno en la plataforma.

**Tamaño:** 90 x 90px

<ArrowRight />

## Iconos para estados vacíos

Estos iconos se muestran en la vistas donde aún no se ha generado el contenido.

<ArrowRight />
